# Project Overview

This project performs analysis related to cities and airports. It processes data from various sources, including city information and airport data, to generate sentiment scores and insights.

## Data

The `data/` directory contains the input datasets used for the analysis, including:
- `airports.csv`: Information about airports.
- `cities.csv`: Information about cities.
- `routes.csv`: Data on routes connecting airports/cities.
- `city_wikis/`: Text files containing information about various cities, likely used for sentiment extraction.

## Results

The `results/` directory stores the output of the sentiment analysis, including:
- `airport_city_sentiments.csv`: Sentiment scores related to airports and cities.
- `airport_sentiment_results.csv`: Overall sentiment results for airports.
- `all_city_sentiments.csv`: Comprehensive city sentiment data.
- `city_sentiments.csv`: Sentiment scores specific to cities.
- `sentiment_scores_custom.csv`: Custom sentiment scores.
